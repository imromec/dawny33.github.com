You are exploring the laboratory closely. You discover that I have developed things like:

> Codejudge
An online judge that you can use to automate coding competitions. Featuring a super-responsive design, it can be easily setup to run from your system. The judge is multi-threaded and supports major languages like C, C++, Java and Python.
Get it or know more from the homepage or github project page.

> Panther
A cross-platform audio/video file converter, powered by ffmpeg. It is super-fast, and has been reviewed across multiple sites for its simplicity of use. Apart from converting formats it can also be used to reduce the size of music files, joining and splitting files. It features a javascript engine to create user scripts, to automate batch processing tasks and add unlimited functionality to it.
Get it or know more from the homepage or github project page.

> Mirrorify
Generate a single URL for your mirror sites. You get the primary URL if its server is up or the second URL otherwise. Clean and simple. Powered by Node.js and Redis.
Find Mirrorify at github.com/sankha93/mirrorify.

> WeTunes
A social music player for Android. I had developed a full social network for it. Features include sending friend requests, viewing what songs friends are listening to, suggesting songs. The app could fetch lyrics automatically from the internet, and lookup videos from Youtube. It has been discontinued.
The source for the app is at github.com/sankha93/wetunes-android and server is at github.com/sankha93/wetunes-server.
